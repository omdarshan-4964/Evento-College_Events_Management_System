## Print-Friendly Version

**Project:** Evento - College Event Management System  
**Live Demo:** https://evento-zeta-nine.vercel.app/  
**Setup Time:** 5 minutes with demo data  

**Demo Credentials:**
- Student: `student@demo.com` / `demo123`
- Club Admin: `club@demo.com` / `demo123`
- Super Admin: `admin@demo.com` / `demo123`

```bash

npm run dev     # In both server and client folders
```

- 3-tier role-based access control
- Automated venue conflict detection
- Interactive calendar view
- Analytics dashboard
- Dark mode support
- Mobile responsive

**Tech Stack:** React + Node.js + MongoDB + Express + JWT + Tailwind CSS
